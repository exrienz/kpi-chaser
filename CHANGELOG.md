# Changelog

## 0.2.0 - 2026-03-12

- Redesigned the dashboard UI with clearer quarter, KPI, and sub-KPI hierarchy.
- Added an authentication gate so dashboard data is only available after login or registration.
- Added confirmation workflows for destructive actions, including protected KPI deletion.
- Added a password-confirmed `Reset all progress` flow with server-side verification and reset audit logging.
- Added a yearly checkpoint view that aggregates quarterly KPI and sub-KPI reporting data.
